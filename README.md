# Bug Bounty Automation Toolkit

## Overview
This project automates reconnaissance workflow for bug bounty hunting and penetration testing.

## Features
- Subdomain Enumeration
- Live Host Detection
- Port Scanning
- URL Collection
- Vulnerability Scanning (Nuclei)

## Tools Used
- subfinder
- httpx
- nmap
- waybackurls
- nuclei

## Usage
./recon.sh example.com

## Output Structure
Results are stored inside:
output/<target-domain>/

## Disclaimer
This tool is intended for authorized security testing and educational purposes only.
